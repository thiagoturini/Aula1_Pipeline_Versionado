# Pipeline de Coleta de Filmes

Pipeline de dados para coletar informações de filmes da API do TMDb.

## Estrutura

- `models.py`: Classes de dados
- `config.json.example`: Exemplo de configuração
- `01_setup.ipynb`: Configuração inicial
- `02_coleta_dados.ipynb`: Coleta de dados
- `03_estruturacao.ipynb`: Estruturação com classes
- `04_versionamento.ipynb`: Versionamento com Git

## Como Usar

1. Copie `config.json.example` para `config.json`
2. Adicione sua API key do TMDb
3. Execute os notebooks na ordem

## Requisitos

- Python 3.11+
- requests

## API Key

Obtenha uma API key gratuita em: https://www.themoviedb.org/
